# 🎓 Ultimate Student Helper App

A **Python & Streamlit web app** designed for students to **predict scores, calculate CGPA, generate project ideas, practice questions, and share resources**. Built to help students organize academic life and excel in school projects.

---

## **Features**

### ✅ Phase 1 (Implemented)
- **Score Predictor**: Predict your score based on hours studied using machine learning.
- **CGPA Calculator**: Calculate your CGPA for University or Polytechnic using your course scores and units.
- **Visual Results**: Charts for performance and grade points.

### 🚀 Phase 2 (Planned / In Progress)
- **Login/Register System**: User-specific data storage.
- **Project Helper**: Suggest final year project topics based on department and interest.
- **Practice Questions**: Generate questions from uploaded course PDFs.
- **Resource Sharing**: Upload/download notes and mini-projects between students.
- **Multiple Semester Tracking**: Track CGPA across semesters.

---

## **Technologies Used**
- Python 3
- Streamlit
- pandas
- scikit-learn
- matplotlib
- SQLite (for user and CGPA data storage)
